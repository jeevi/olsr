Module main

source code

This is the main program. upon executing "python main.py" on the console, this program, reads the first line from the input file named 'input'. The first line must be the number of processes [nodes] in the network. This is needed to initialize the 2-D list of size n*n [n - number of nodes], which contains a mapping of links between different nodes. Next, ths control enters an infinite loop, in which the 'edge' information between two nodes is read, the control breaks out of the loop when an EOF is encountered in the file 'input'. the link information is stored in the 'edges' list. From here, the method creaetProcesses is called , it exists in the file Node.py
